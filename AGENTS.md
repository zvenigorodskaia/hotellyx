# Hotellyx

Hotellyx is a mobile-first web app for hotels to handle guest requests and services without front desk calls.

## Goal
Ship a working MVP to pilot in a real hotel.

## Guest (no login, via QR)
- Entry via /r/[room_token]
- Room status (Do Not Disturb / Cleaning)
- Quick actions (towels, cleaning, issue)
- Services list
- Submit request
- View request status

## Staff
- Login via email magic link
- Requests dashboard
- Update request status (new, accepted, in_progress, done)

## Tech
- Next.js (App Router)
- Tailwind CSS
- Supabase (DB + Auth)
- Vercel deployment

## Constraints
- No payments
- No PMS integrations
- One hotel
- One language (EN)
