# Hotellyx

Minimal Next.js scaffold for the Hotellyx repository using the App Router and Tailwind CSS.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Local Setup

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open:

[http://localhost:3000](http://localhost:3000)

## Available Routes

- `/`
- `/r/[token]`
- `/staff/login`
- `/staff`

## Environment Variables

Create a `.env.local` file in the project root when needed:

```env
# Example placeholders
NEXT_PUBLIC_APP_NAME=Hotellyx
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```
